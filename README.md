# AI-Based Diet Recommendation System

## Project Description

The AI-Based Diet Recommendation System is a web application developed using Python and Streamlit that provides personalized diet plans based on user information such as age, gender, height, weight, food preference, and fitness goals.

## Objective

- Calculate Body Mass Index (BMI).
- Determine the user's health category.
- Recommend personalized diet plans.
- Generate downloadable PDF reports.

## Key Features

- User-friendly interface.
- BMI calculation and health assessment.
- Personalized diet recommendations.
- Vegetarian and non-vegetarian meal plans.
- PDF report generation.
- Downloadable diet reports.

## Technologies Used

- Python
- Streamlit
- ReportLab
- JSON

## Modules Used

- Streamlit
- ReportLab
- JSON
- Datetime

## System Workflow

1. User enters personal details.
2. System calculates BMI.
3. Health category is identified.
4. Diet plan is generated.
5. PDF report is created.
6. User downloads the report.

## Advantages

- Easy to use.
- Personalized recommendations.
- Downloadable reports.
- Promotes healthy eating habits.

## Future Enhancements

- AI/ML-based recommendations.
- Disease-specific diet plans.
- Fitness tracker integration.
- Email report delivery.
- Multi-language support.

## Conclusion

The system provides personalized diet recommendations based on user information and fitness goals, helping users maintain a healthy lifestyle.
