# AI-Diest-Recommendation-System

Project Description
The AI-Based Diet Recommendation System is a web application developed using Python and Streamlit that provides personalized diet plans based on user information such as age, gender, height, weight, food preference, and fitness goals. The system calculates the user's Body Mass Index (BMI), determines their health category, and recommends suitable meal plans to support weight loss, weight gain, or maintenance goals.
Objective
The main objective of this project is to help users make healthier dietary choices by generating customized diet recommendations based on their physical and lifestyle requirements.
Key Features
User-friendly interface for entering personal details.
Automatic BMI calculation and health assessment.
Personalized diet recommendations.
Support for vegetarian and non-vegetarian meal plans.
PDF report generation.
Downloadable diet recommendation report.
